Дискретные структуры 2013—2014:
задания по программированию на Python

Правила сдачи

  • Суммарно за сдачу заданий по программированию нельзя получить более 20 баллов.
  • Каждая задача должна быть прислана на проверку заранее и, если она предварительно зачтена, нужно её очно защитить (ответить на вопросы лектора). Крайний срок защиты заданий: 20 июня.
  • Задания обязательно писать на языке Python 3 (программы на других языках, в частности, на Python 2, не принимаются). Если Вы не знаете Python, не переживайте. Основы языка учатся быстро, и в жизни он может пригодиться. Также это один из самых популярных языков для научных вычислений и прототипирования. Один из плюсов языка: стандартные математические структуры (множества, кортежи, списки) являются полноценной частью стандарта языка, а не надстройкой.
  • Программы присылаются лектору на почту. Семинаристы программы не принимают.
  • Решение каждой задачи высылается в отдельном письме.
  • В письме нужно указать свои имя, фамилию и номер академической группы. К письму нужно прикрепить единственный скрипт, решающий задачу.
  • Не забудьте про общие требования к оформлению писем.
  • Программа должна быть откомментирована. Комментарии допускаются на русском и английском языке. Комментарии транслитом не допускаются.
  • К некоторым программам может требоваться отчёт, оформленный в TeX. Можно использовать, например, среду MiKTeX или облачный сервис ShareLaTeX.

Задачи

Py1. Решение рекуррентных соотношений третьего порядка

Стоимость задачи: 8 бонусных баллов. При использовании библиотек NumPy/SciPy стоимость понижается до 6 баллов.

Требуется написать программу, которая по рекуррентному соотношению (не более чем) третьего порядка выдаёт формулу общего решения этого соотношения в LaTeX-формате.

На вход программе подаётся строка следующего вида:

  • коэффициент идентификатор_{индекс} + коэффициент идентификатор_{индекс} + коэффициент идентификатор_{индекс} + коэффициент идентификатор_{индекс} = 0

Некоторые из членов могут отсутствовать. Также могут отсутствовать коэффициенты, равные единице. Идентификатор однобуквенный, переменная в индексе тоже. Примеры корректных входных строк:

  • 15 a_{n} + 8 a_{n-2} - 99 a_{n-3} = 0
  • 8 V_{s-1} + V_{s+2} + V_{s} = 0

Программа должна выдать формулу для общего члена последовательности следующего вида:

  • $идентификатор_индекс = выражение$

        Здесь выражение — это запись в TeXовской нотации. Например:

  • $V_s = const_1\cdot 2^s + (const_2 s+const_3)\cdot (-4)^s$

Никаких алгебраических упрощений при получении ответа делать не нужно. Например, не требуется сокращать дроби. Ответ должен быть точным, без округлений.

О том, как решать кубические уравнения, можно прочитать на Википедии.

Py2. Построение производящей функции по рекуррентному соотношению

Стоимость задачи: 4 бонусных балла.

Требуется написать программу, которая по линейному рекуррентному соотношению выдаёт производящую функцию соответствующей последовательности.

Формат входных данных программы идентичен предыдущей задаче, но порядок рекуррентного соотношения может быть любым, в частности, большим трёх.

На выходе программы должно быть выражение в ТеХовской нотации для производящей функции. Пример:

  • $f(x) = \frac{3 x^2 + 7 x - 5}{x^3 + x}$

Py4. Построение «гиперграфа Альсведе—Хачатряна»

Стоимость задачи: 3 бонусных балла.

Требуется написать программу, которая для заданных  выводит список всех рёбер какого-нибудь экстремального (по числу рёбер) k‑однородного t‑пересекающегося гиперграфа на множестве вершин . Каждое ребро на новой строке, в формате

  • <вершина1> <вершина2> <вершина3> … <вершинаk>

(без угловых скобок).

Py5. Построение рамсеевского графа из теоремы Франкла—Уилсона

Стоимость задачи: 5 бонусных баллов.

Требуется написать программу, которая для заданного  выводит произвольные различные 10000 рёбер графа. Ребра нужно выводить по одному на строке в формате

  • <вершина1> <вершина2>

где вершина (т.е. вектор длины ) задаётся в формате

  • <координата1><координата2>…

Например, при  ребро могло бы выглядеть так:

  • 11110000 11000110

Py6. Построение матрицы Заранкевича

Стоимость задачи: 6 бонусных баллов.

Требуется написать программу, которая для заданного  строит матрицу (по обоснованному на лекции методу) размера , в которой  единиц и которая не содержит «плохих» 2×2-подматриц. Вход программы: число p. Выход: матрица. Каждая строка матрицы выводится на отдельной строке экрана, а отдельные элементы разделяются пробелами.

Py7. Разложение ч.у.м. на антицепи

Стоимость задачи: 6 бонусных баллов.

Требуется написать программу, которая строит разложение заданного ч.у.м. на минимально возможное число цепей. На вход программы подаётся диаграмма Хассе частично упорядоченного множества в виде набора строк в формате:

  • меньший_элемент больший_элемент

Элементы ч.у.м. — произвольные строки, не содержащие пробелов. Напомним, что в диаграмме Хассе мы указываем только пары непосредственно следующих друг за другом элементов (между которыми нельзя вставить никакого другого элемента ч.у.м.).

Программа должна выдавать на выходе антицепи (по одной на каждой строке), на которые разлагается ч.у.м.. Элементы в антицепях указываются через пробел.

Например, на входе

  • ds 2
  • 2 ***
  • ds X
  • X 5

программа должна выдавать разложение на три антицепи:

  • ds
  • 2 X
  • *** 5

Py8. Вычисление функции Мёбиуса на ч.у.м.

Стоимость задачи: 5 бонусных баллов.

Требуется написать программу, которая вычисляет функцию Мёбиуса на заданном частично упорядоченном множестве. На вход программы подаётся диаграмма Хассе частично упорядоченного множества в том же формате, как и в задаче Py7.

Программа должна выдавать список значений функции Мёбиуса на всех парах сравнимых элементов. Например, на входе из задачи Py7 программа должна выдавать такой список (порядок, в котором выводятся значения функции Мёбиуса, не важен):

  • ds ds 1
  • ds 2 -1
  • 2 2 1
  • 2 *** -1
  • ds *** 0
  • …

Py9. Вычисление VC-размерности семейства множеств

Стоимость задачи: 5 бонусных баллов.

Дано семейство множеств в формате:

  • <элемент1множества1> <элемент2множества1> …
  • <элемент1множества2> <элемент2множества2> …
  • …

Элементы — произвольные строки, не содержащие пробелов. Программа должна вычислять размерность Вапника—Червоненкиса заданного семейства и на отдельной строчке выводить соответствующее дробящееся множество. Например, для входа

  • 1 2 3
  • 1 3 5
  • 2 4 5
  • 7

программа должна выдавать

  • 2
  • 1 2

или

  • 2
  • 3 5
